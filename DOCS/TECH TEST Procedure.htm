 Instruction #3 (Base functions for Well Info components)

 

TT29090, TT29089, TT29088

 

FIELD – Field Name

WELL – Well Name

TRFReport – TRF File with Global Report

COMP_NAME – name of component]

Etalon Data File – File with Etalon Data

Etalon Report – Filed with Etalon Report

TRFComponent – TRF File with component

 

 

1.       Add FIELD using System Configuration Explorer

2.       Add WELL for FIELD using Well Explorer

3.       Quick Import ( TRFReport )

4.       Edit Component

a.       Launch Well Explorer using main menu

b.      Wait For Well Explorer to appear

c.       Select Tree Item ("/All Wells (by Field)/FIELD (*)/WELL")

d.      Select Column “Name” using Screen setup dialog

e.      Select COMP_NAME from list

f.        Press Button "Edit Data" from action zone

g.       Wait For Screen of component to appear

5.       Add And Verify Data

a.       Set Active Screen

b.      Press Add button from toolbar

c.       Press Add button from toolbar

d.      Press Add button from toolbar

e.      List Paste Data using Ctrl – Home and Ctrl - V

f.        Select All data from list using Ctrl – Home and Ctrl – C

g.       Get Text With Header using pop-up menu

h.      List Verify Text with Etalon Data File

i.         Press button Save from toolbar

j.        Close Screen

k.       Reset System Messages

6.       Edit Component. Repeat Step 4

7.       Verify Data With Report

a.       Set Active Screen

b.      Select All Data from list using Ctrl – Home and Ctrl – A

c.       Press Global Output button from toolbar

d.      Verify Report with Etalon Report using export to (*.txt)

e.      Close Screen

f.        Reset System Messages

8.       Export

a.       Launch Well Explorer using main menu

b.      Wait For Well Explorer to appear

c.       Select Tree Item ("/All Wells (by Field)/FIELD (*)/WELL")

d.      Select Column “Name” using Screen setup dialog

e.      Select COMP_NAME from list

f.        Press Button "Express Export" from action zone

g.       Wait For Express Export Wizard to appear

h.      Quick Export (TRFComponent)

i.         Close Well Explorer

9.       Edit Component. Repeat Step 4

10.   Delete All Data

a.       Set Active Screen

b.      Select All Data from list using Ctrl – Home and Ctrl – A

c.       Press Delete button from toolbar

d.      Wait For Confirm dialog to appear

e.      Press button Yes from this dialog

f.        Select All Data from list using Ctrl – Home and Ctrl – A for Check if Data Empty

g.       Press button Save from toolbar

h.      Close Screen

11.   Import

a.       Launch Well Explorer using main menu

b.      Wait For Well Explorer to appear

c.       Select Tree Item ("/All Wells (by Field)/FIELD (*)/WELL")

d.      Select Column “Name” using Screen setup dialog

e.      Select COMP_NAME from list

f.        Press Button "Express Import" from action zone

g.       Import TRF (TRFComponent, {WELL, “Replace”})

h.      Close Well Explorer

12.   Edit Component. Repeat Step 4

13.   Verify Data

a.       Set Active Screen

b.      Select All Data from list using Ctrl – Home and Ctrl – A

c.       Get Text With Header using pop-up menu

d.      List Verify Text with Etalon Data File

e.      Close Screen

14.   Edit Component. Repeat Step 4

15.   Verify Data With Report. Repeat Step 7.

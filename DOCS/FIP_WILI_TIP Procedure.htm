b)     On toolbar click “Import from Excel”

c)     Press “Next” button 2 times

d)     For “PCF File Name” group box enter: "S:\FIP_WILI_TIP\DATA\WILI Well Services.mig"

e)     For “Select a specific discreet file as the data source” group box enter: "S:\FIP_WILI_TIP\DATA\B 33-96, Job report BST test.xls"

f)      Press “Next” button.

g)     On list control set check in “Use” column for “Events” row.

h)     Press “Next” button 2 times and then “Finish”.

i)      Wait for progress bar to disappear.

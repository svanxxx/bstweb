Instruction #1 (Map verification)

Parameters:

                Shelf Name: string – indicates what Shelf of sidebar should be selected

                Button Name: string – indicates what button should be pressed to run map on Shelf of OMV sidebar

                Settings Name: string – indicates what settings should be selected from combobox on first screen of Fieldpro QEZ Wizard

                File Name: string – indicates what file name for saving map

                Should be closed: check –can be Yes or No indicates if map Should be closed at the end

          

1.       Select Shelf on sidebar with name Shelf Name

2.       Click button Button Name

3.       On Fieldpro QEZ Wizard select “Load previous Settings from Database” radiobutton

4.       Select settings with Settings Name from combobox Settings Name

5.       Click button Finish

6.       After finishing loading map click button 100% zoom from map toolbar 

7.       Set landscape orientation for the page on page setup screen

8.       Print to pdf (File Name) and convert to png

9.     If Should be closed = Yes then close map without saving

 

Instruction #2 (Map verification zoom)

Parameters:

Top Left Latitude: number - indicates Latitude of Top Left Extent point in Map Extent

Top Left Longitude: number - indicates Longitude of Top Left Extent point in Map Extent

Bottom Right Latitude: number - indicates Latitude of Bottom Right Extent point in Map Extent

Bottom Right Longitude: number - indicates Longitude of Bottom Right Extent point in Map Extent

Coordinate System string: indicates map coordinate system

File Name: string – indicates what file name for saving map

Should be closed: check –can be Yes or No indicates if map Should be closed at the end

Coordinate System: Geodetic - Base WGS 84

 

Conditions:

For run this instruction you should have Map opened already

 

1.       Click button Map Properties from toolbar

2.       On Map Properties tab click Edit button for Top Left Extent point in Map Extent  group

3.       Put Top Left Latitude: indicates Latitude of Top Left Extent point in Map Extent

4.       Put Top Left Longitude: indicates Longitude of Top Left Extent point in Map Extent            

5.       Click Ok

6.       On Map Properties tab click Edit button for Top Left Extent point in Map Extent  group

7.       Top Bottom Right Latitude: indicates Latitude of Bottom Right Extent point in Map Extent

8.       Top Bottom Right Longitude: indicates Longitude of Bottom Right Extent point in Map Extent            

9.       Click Ok

10.   Click Ok

11.   Click button 100% zoom from map toolbar

12.   Set landscape orientation for the page on page setup screen

13.   Print to pdf (File Name) and convert to png

14.   If Should be closed = Yes then close map without saving
